# PHP_sleep

An assignment practising OOP PHP

Requirements:

- Create a Web form that allows a user to enter their first name, age, and how many hours they sleep at night
- Use a constructor on the class you create to hold the functions and variables
- Specify the number of hours in a day in a constant
- When they click a "Calculate" button, use an Object-oriented PHP function to calculate how many years of their life they have slept and then display a messafe in the followong format
  "Hey Julia, you have managed to sleep for 19 years of your life!"
